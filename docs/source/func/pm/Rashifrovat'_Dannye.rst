
РасшифроватьДанные
===================================

**Синтаксис:**

РасшифроватьДанные(Параметры)

**Описание:**

Возвращает расшифрованные данные документа.

**Входные параметры:**

**Параметры** (тип Структура)

      **Элементы структуры:**

      * **ЗашифрованныеДвоичныеДанные** (тип ДвоичныеДанные) — зашифрованный файл документа.

**Возвращает:**

ДвоичныеДанные
