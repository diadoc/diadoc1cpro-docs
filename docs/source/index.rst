
Модуль Диадока для 1С
=====================

Модуль предназначен для отправки, получения и просмотра электронных документов, передаваемых через систему электронного документооборота Диадока.

Компании мгновенно могут отправлять документы своим клиентам, поставщикам, партнерам, удаленным подразделениям прямо из 1С. Это могут быть как документы в формате ФНС, так и произвольные файлы.

Документы в формате ФНС в сервисе подписываются электронной подписью и обладают юридической силой. Подписание прочих документов настраивается в соответствии с бизнес-процессами компании.
