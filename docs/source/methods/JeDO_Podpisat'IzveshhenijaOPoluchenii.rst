ЭДО_ПодписатьИзвещенияОПолучении
===================================

**Синтаксис:**

ЭДО_ПодписатьИзвещенияОПолучении

**Описание:**

Добавляет двоичные данные подписи в каждый элемент структуры входных параметров.

**Входные параметры:**

      * **ИоПы** (тип Структура) — сформированные извещения о получении электронных документов.
