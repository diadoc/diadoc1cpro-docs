ЭДО_ПодготовитьИОтправитьОтветНаДокумент
=============================================

**Синтаксис:**

ЭДО_ПодготовитьИОтправитьОтветНаДокумент

**Описание:**

Формирует и отправляет ответный титул.

**Входные параметры:**

      * **ДокументЭДО** (тип СправочникСсылка.Диадок_Документы) — ссылка на описание электронного документа в базе 1С.
