ДатаXML_в_Дата1С
=============================================

**Синтаксис:**

ДатаXML_в_Дата1С

**Описание:**

Переводит дату из формата Строка в формат XML.

**Входные параметры:**

      * **ДатаXML** (тип Строка) — строковое представление даты (YYYY.MM.DD).

**Возвращает:**

Дата в формате XML.
